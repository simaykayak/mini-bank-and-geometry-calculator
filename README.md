# Mini Bank and Geometry Calculator
A beginner-level C project that combines financial calculations, geometry operations, type casting, memory size checks, and logical comparisons.

## Features
- Simple and compound interest calculation
- Circle area and circumference calculation
- Hypotenuse calculation
- Integer and decimal division
- Memory size display with `sizeof`
- Relational and logical operator examples

## Topics Covered
- `printf`, `scanf`, `getchar`, `putchar`
- `int`, `float`, `double`, `char`
- `pow()` and `sqrt()`
- Type casting
- Assignment and increment operators
- Relational and logical operators
